# 📦 배포 가이드 (최종 사용자용)

이 문서는 **개발자가 아닌 일반 사용자**를 위한 간단한 사용 가이드입니다.

---

## 🎯 프로그램 소개

**AI 프롬프트 생성기**는 이미지를 분석하여 Midjourney, DALL-E 같은 AI 이미지 생성 도구에 사용할 고품질 프롬프트를 자동으로 만들어주는 프로그램입니다.

### ✨ 특징
- ✅ Python 설치 불필요
- ✅ 완전 무료
- ✅ 압축 해제 후 바로 실행
- ✅ 간단한 사용법

---

## 📥 설치 방법

### 1단계: 다운로드
`prompt-maker-v1.0.zip` 파일을 다운로드합니다.

### 2단계: 압축 해제
원하는 폴더에 ZIP 파일을 압축 해제합니다.

**예시:**
```
C:\프로그램\prompt-maker-v1.0\
```

### 3단계: 폴더 구조 확인

```
prompt-maker-v1.0/
├── 프롬프트생성기.exe      ⬅️ 이 파일을 실행!
├── _internal/              (필수 파일, 삭제 금지)
├── .env.example            (API Key 설정 파일)
└── 사용법.txt              (상세 가이드)
```

---

## 🔑 API Key 설정 (최초 1회)

### 1. API Key 발급

1. https://aistudio.google.com/app/apikey 접속
2. Google 계정으로 로그인
3. **"Create API Key"** 버튼 클릭
4. 생성된 API Key를 복사

> 💰 **완전 무료!** (하루 1,500번 사용 가능)

### 2. 프로그램에 API Key 입력

**방법 1: .env 파일 사용 (권장)**

1. 폴더에서 `.env.example` 파일 찾기
2. 파일 이름을 **`.env`** 로 변경
3. 메모장으로 열기
4. 다음과 같이 수정:
   ```
   GEMINI_API_KEY=여기에_복사한_API_Key_붙여넣기
   ```
5. 저장 후 닫기

**방법 2: 프로그램에서 직접 입력**

1. 프로그램 실행 후 API Key 입력란에 붙여넣기
2. "테스트" 버튼으로 연결 확인

## 🚀 사용 방법

### 1. 프로그램 실행
`프롬프트생성기.exe` 파일을 더블클릭합니다.

### 2. 이미지 선택
- "파일 선택..." 버튼 클릭
- 참고할 이미지 선택 (JPG, PNG, WEBP)
- 이미지 1은 필수, 이미지 2는 선택사항

### 3. 텍스트 입력
원하는 스타일이나 장면을 설명합니다.

**예시:**
- "두 양갈래 소녀들이 하이파이브하고 있는 지브리 스타일로"
- "사이버펑크 도시 야경, 네온사인"
- "수채화 풍경, 따뜻한 색감"

### 4. 프롬프트 생성
- "프롬프트 생성하기" 버튼 클릭
- 1-10초 대기 (AI 분석 중)

### 5. 결과 사용
- **"복사" 버튼**: 생성된 프롬프트를 클립보드에 복사
- **"JSON 저장" 버튼**: 파일로 저장
- 복사한 프롬프트를 Midjourney, DALL-E 등에 붙여넣기

## ❓ 자주 묻는 질문

### Q1. 프로그램이 실행되지 않아요

**확인 사항:**
1. `_internal` 폴더가 같은 위치에 있는지 확인
2. Windows Defender가 차단하는지 확인
3. 백신 프로그램 예외 설정 추가

**해결 방법:**
- 프로그램 우클릭 → "관리자 권한으로 실행"
- Windows Defender 예외 설정:
  1. Windows 보안 열기
  2. 바이러스 및 위협 방지
  3. 설정 관리
  4. 제외 항목 추가
  5. 폴더 선택 → `prompt-maker-v1.0` 폴더 추가

---

### Q2. "API 연결 실패" 오류가 나요

**확인 사항:**
1. `.env` 파일이 있는지 확인 (`.env.example`이 아님!)
2. API Key가 올바르게 입력되었는지 확인
3. 인터넷 연결 확인
4. 하루 사용 횟수(1,500회) 초과 여부 확인

---

### Q3. 이미지가 너무 크다고 나와요

**해결 방법:**
- 이미지를 10MB 이하로 줄여주세요
- 온라인 이미지 압축 도구 사용
- 이미지 편집 프로그램으로 해상도 조정

---

### Q4. 프롬프트가 영어로만 나와요

**정상입니다!**
- Midjourney, DALL-E 등 AI 이미지 생성 도구는 영어 프롬프트를 권장합니다
- 더 정확하고 고품질의 결과를 얻을 수 있습니다

---

### Q5. Python을 설치해야 하나요?

**아니요!**
- 이 프로그램은 독립 실행형입니다
- Python이나 다른 프로그램 설치 불필요
- 압축 해제 후 바로 실행 가능

## 🆘 문제 발생 시

### 1단계: 프로그램 재시작
- 프로그램 종료 후 다시 실행

### 2단계: 폴더 재확인
- 모든 파일이 제대로 압축 해제되었는지 확인
- `_internal` 폴더 존재 여부

### 3단계: .env 파일 재확인
- 파일 이름이 `.env`인지 확인 (`.env.example`이 아님)
- API Key 앞뒤 공백 제거
- 따옴표 없이 입력했는지 확인

### 4단계: API Key 재발급
- 기존 API Key 삭제
- 새로운 API Key 발급 후 재설정
